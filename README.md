# AgentFoot

Simulation mobile-first d'agent de joueurs de football.

## Lancer en local

```bash
npm install
npm run dev
```

## Build production

```bash
npm run build
```

Le build de production est généré dans `dist/`.

## Déploiement Vercel

- Framework: Vite
- Build command: `npm run build`
- Output directory: `dist`
